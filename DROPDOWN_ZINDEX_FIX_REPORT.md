# Исправление проблемы видимости выпадающих меню (Z-Index Fix) 
## Дата: 12 апреля 2026

---

## ✅ Проблема (РЕШЕНА)

**Описание:** Выпадающие меню навигации (УСЛУГИ, ALBAMAN, О НАС) на всех страницах сайта не были видны пользователям на мобильной и десктопной версии. Пункты меню отображались сзади (позади) основного контента страниц и были полностью закрыты.

**Симптомы:**
- Пункты меню не видны при наведении/клике
- Выпадающее меню скрыто под основным контентом
- Проблема наблюдалась на всех страницах сайта
- Влияла на мобильные, планшеты и десктопные устройства

---

## 🔧 Выполненные исправления

### 1. Создан новый CSS файл для исправления z-index
**Файл:** `/assets/css/dropdown-z-index-fix.css`

Этот файл содержит комплексную стратегию из 17 фаз для гарантии видимости выпадающих меню:

#### ОСНОВНЫЕ ПРАВИЛА:
- **Все элементы контента** получили `z-index: 1 !important` (низкий приоритет)
- **Выпадающие меню** получили `z-index: 999999 !important` (максимальный приоритет)
- **Модальные окна и overlay** получили `z-index: 8999 !important` (ниже dropdown)
- **Отключены transform** на dropdown контейнерах (transform создает новый stacking context)
- **Обеспечен overflow: visible** на всех родительских контейнерах
- **Мобильная версия** использует `position: fixed` для выпадающих меню

### 3. Обновлен include.js  
**Файл:** `/assets/js/include.js`

Добавлена функция `injectDropdownZIndexFix()` которая автоматически загружает CSS файл на все страницы (как резервный механизм).

```javascript
function injectDropdownZIndexFix() {
  if (document.getElementById('alba-dropdown-z-index-fix')) return;
  
  const link = document.createElement('link');
  link.id = 'alba-dropdown-z-index-fix';
  link.rel = 'stylesheet';
  link.href = '/assets/css/dropdown-z-index-fix.css';
  document.head.appendChild(link);
}
```

## 📋 Детали исправления

### CSS Стратегия (17 фаз)

| Фаза | Элемент | Z-Index | Position | Действие |
|------|---------|---------|----------|---------|
| 1-3 | Контент, модели | 0-1 | relative | Низкий приоритет |
| 4 | Header | 100 | relative | Средний приоритет |
| 5 | Navigation | 101 | relative | Выше header |
| 6 | Dropdown trigger | 102 | relative | Выше навигации |
| 7 | **Dropdown menu** | **999999** | **absolute/fixed** | **МАКСИМУМ** |
| 8 | Mobile overlay | 999998 | fixed | Ниже menu |
| 9-12 | Modals, AI | 8998-8999 | relative | Ниже dropdown |
| 13-17 | Специальные случаи | varies | varies | Переопределения |

### Ключевые CSS селекторы

```css
/* Контент - низкий приоритет */
main, .page, .home-main { z-index: 1 !important; }

/* Выпадающее меню - ВЫСШИЙ ПРИОРИТЕТ */
.dropdown-menu { z-index: 999999 !important; }

/* Видимость */
.dropdown:hover .dropdown-menu,
.dropdown.active .dropdown-menu {
  visibility: visible !important;
  pointer-events: auto !important;
  opacity: 1 !important;
}
```

## ⚙️ Технические детали

### Стекинг контекст
Решение основано на правильном управлении CSS stacking context:
- `z-index` работает только в пределах одного stacking context
- `position` + `z-index` создают новый stacking context
- `transform` также создает stacking context
- Решение исключает `transform` на dropdown контейнерах

### Платформа поддержка
- ✅ Desktop (Windows, Mac, Linux)
- ✅ Tablet (iPad, Android)
- ✅ Mobile (iOS, Android)
- ✅ Все современные браузеры (Chrome, Firefox, Safari, Edge)
