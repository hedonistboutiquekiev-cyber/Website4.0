## 🚀 Использование

### Для конечных пользователей
1. Нажмите на кнопку **телефонной трубки** (голос) внизу справа
2. Разрешите доступ к микрофону
3. Говорите четко
4. Получите ответ от Albamen

### Для разработчиков

Запустите диагностику:
```javascript
VoiceDiagnostics.runFullDiagnostics()
```

Проверьте отдельные компоненты:
```javascript
VoiceDiagnostics.checkWorkerConnection()  // Проверить Worker
VoiceDiagnostics.testMicrophone()         // Тест микрофона
VoiceDiagnostics.testSpeechSynthesis()    // Тест синтеза
```

## 📞 Поддержка

Если голосовой чат не работает:

1. **Проверьте браузер** - Chrome, Firefox, Safari, Edge?

3. **Запустите диагностику** - `VoiceDiagnostics.runFullDiagnostics()`
4. **Проверьте микрофон** - `VoiceDiagnostics.testMicrophone()`
